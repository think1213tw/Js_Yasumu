# Js_Yasumu
使用技術:axios套件進行資料串接
        Bootstrap+SCSS規劃響應式版型
        
前台:https://think1213tw.github.io/Js_Yasumu/

1.Get-產品展示(產品分類、加入購物車)

2.Post-預定資料(表單驗證)

3.Delete-單筆品項

4.Delete-所有品項

5.Patch-購物車做數量加減

後台:https://think1213tw.github.io/Js_Yasumu/admin.html

1.可切換圖表查看營收

2.Get-訂單列表

3.Put-訂單狀態

4.Delete-單筆訂單

5.Delete-全筆訂單
